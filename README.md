slim-mvc
========

Support MVC for Slim Framework

## Installation

Plates is available via Composer:

```json
{
    "require": {
        "slim-extensions/mvc": "dev-master"
    }
}
```

## Example of usage
See [slim-extensions-example](https://github.com/dimkof/slim-extensions-example)
